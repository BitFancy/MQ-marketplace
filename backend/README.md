## Installation

1) Clone the repository
```
git clone https://github.com/ground-creative/nft-market-backend
```

2) Change environment variables in env.sample file and rename it to .env

## Usage

Intall node modules:
```
npm install
```

Run node server:
```
node index.js
```

run with nodemo:
```
npm install -g nodemon
nodemon index.js
```
